Reporting a Vulnerability

The security of Prakash Knowledge Hub is important to us. If you discover a security vulnerability, please report it responsibly.

Contact

- Email: prakashphiroda@gmail.com
- GitHub Issues: Please do not report security vulnerabilities in public GitHub Issues.
- GitHub Profile: https://github.com/prakashphiroda314

What to Include

Please include:

- A clear description of the vulnerability.
- Steps to reproduce the issue.
- The affected page or file.
- Screenshots or proof of concept (if available).
- Any suggested fix (optional).

Response Time

- Initial response: Within 72 hours.
- Status updates: As progress is made.
- Fix timeline: Depends on the severity of the issue.

Thank you for helping make Prakash Knowledge Hub more secure for everyone.
